# MIRA | Метео-обеспечение полётов БВС

**Версия:** 0.1.5.0

Система метеорологического обеспечения и анализа рисков для беспилотных воздушных судов.

## 🚀 Возможности

- ✅ Получение данных с Open-Meteo API
- ✅ Анализ метеоусловий по 4 высотам (250, 350, 450, 550 м)
- ✅ Построение маршрутов по нескольким точкам
- ✅ Пространственное распределение рисков вдоль маршрута
- ✅ Вертикальный профиль рисков
- ✅ Анализ: ветер, осадки, видимость, обледенение, турбулентность
- ✅ Экспорт отчётов в PDF
- ✅ Интерактивная карта OpenLayers
- ✅ **Сидя на земле** — коррекция прогноза по фактическим данным с геолокацией

## 🚁 Сидя на земле (новая функция!)

Кнопка **🚩 Сидя на земле** (справа от кнопки «Анализ») позволяет уточнить прогноз Open-Meteo на основе фактических данных с вашей текущей позиции.

### Как использовать:

#### Вариант 1: Автозаполнение из Open-Meteo
1. **Нажмите кнопку 🚩 Сидя на земле**
2. **Нажмите «Моё местоположение»** — форма заполнится данными из Open-Meteo для вашей текущей геопозиции
3. **При необходимости отредактируйте данные** (если есть фактические замеры с датчиков)
4. **Нажмите «Применить коррекцию прогноза»**

#### Вариант 2: Ручной ввод данных с датчиков
1. **Нажмите кнопку 🚩 Сидя на земле**
2. **Введите данные с ваших датчиков:**
   - Ветер (м/с) — средний + порывы
   - Направление ветра (градусы)
   - Температура (°C)
   - Влажность (%)
   - Видимость (км)
   - Нижняя граница облаков (м)
   - Отметьте туман/осадки/снег (если есть)
3. **Нажмите «Применить коррекцию прогноза»**

### Что делает система:

- Сравнивает прогноз Open-Meteo с фактическими данными
- Рассчитывает коэффициенты коррекции (ветер, температура, влажность)
- Переопределяет видимость и облачность при необходимости
- Применяет коррекцию ко всем высотам (250–550 м) с затуханием
- Пересчитывает риски с учётом поправок
- Формирует рекомендации перед взлётом
- Определяет максимально допустимую высоту полёта

### Вкладка «Отчёт» → «С учётом корректировки»

После применения коррекции перейдите на вкладку **«Отчёт»** и нажмите кнопку **«② С учётом корректировки»** — увидите рекомендации с учётом ваших данных.

### Пример:

```
Прогноз Open-Meteo:  ветер 8 м/с, температура +5°C
Факт с датчика:      ветер 11 м/с, температура +3°C
────────────────────────────────────────────────────
Коэффициент ветра:   11/8 = 1.375 (увеличить на 37.5%)
Поправка темп.:      3 - 5 = -2°C

Скорректированный прогноз на 500м:
  Было: 10 м/с, +2°C
  Стало: 13.8 м/с, 0°C  ← риск обледенения!

Рекомендация: Ограничить высоту 400м или отложить полёт
```
